Get variable groups of reduced results.

Usage

getResultVars(data, type = "group")

Arguments

data
[ReducedResultsExperiments] Result data.frame from reduceResultsExperiments.
type
[character(1)] Can be “prob” (prob + pars), “prob.pars” (only problem pars), “algo” (algo + pars), “algo.pars” (only algo pars), “group” (prob + problem pars + algo + algo pars), “result” (result column names). Default is “group”.

Value

[character]. Names of of columns.

Description

Useful helper for e.g. package plyr and such.
